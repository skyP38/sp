## VGA Text Mode - *интерфейс вывода на текста на экран*

# Описание

Драйвер при записи в него, принимает строку текста, которая выводится на "экран".
"Экран" представляет собой таблицу 80 столбцов на 25 строк, в каждой ячейке отображается один символ.
Текст выводится слева направо, при достижении края "экрана" следующий символ переносится на начало
новой строки.
Поддерживаются символы переноса строки(\n) и возврата каретки (\r).
При чтении - драйвер ничего не выводит, но завершает работу без ошибки.
Через ioctl и sysfs:
* Считывается и устанавливается текущая координата курсора
* Считывается и устанавливается цвет текущего и последующих символов
* Считывается и устанавливается цвет фона текущего и последующего символов
* Очистка экрана

Управляющие регистры:
* RW Регистр вывода
* RW Текущая координата курсора
* RW Текущий цвет текста
* RW Текущий цвет фона
* RO Регистр статуса (готов к записи, ошибка)

# Возможности
Выводит переданный текст с гостевой OC в bar2.bin, откуда читает хостовая программа, которая эмулирует экран.

# Команды сборки
Собрать и подключить драйвер на гостевой ОС:
```
make && sudo insmod vgatext.ko
```

Пересобрать и переподключить драйвер на гостевой ОС:
```
make clean && make && ./rerun.sh
```

Собрать и запустить хостовую программу:
```
gcc -o host_vga host_vga.c && ./host_vga
```
